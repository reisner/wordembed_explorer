# wordembed_explorer

A shiny app for exploring word embedding models.

## Setup
Requires a word embeddings file, from word2Vec.

Also requires:
```
library(devtools)
install_github('bmschmidt/wordVectors')
```

## Run app:

From R:

```
shiny::runApp()
```

From the command line:

```
Rscript runapp.R
```
